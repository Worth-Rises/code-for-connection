# UI Components (shadcn/ui)

These are shadcn/ui components — pre-built, customizable React components using Radix UI + Tailwind CSS.

Installed by the admin guild for use across the project. Any guild can import them.

## Adding new components

From the project root:

    npx shadcn@latest add badge table tabs

Full list: https://ui.shadcn.com/docs/components

## Import pattern

    import { Button } from "@/components/ui/button"
    import { Dialog, DialogContent } from "@/components/ui/dialog"
